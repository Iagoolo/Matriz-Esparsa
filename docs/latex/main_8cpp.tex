\doxysubsection{Descrição detalhada}
Programa para manipulação de matrizes esparsas. 

Este programa permite ao usuário ler, imprimir, somar e multiplicar matrizes esparsas. As matrizes são armazenadas em um mapa associativo, onde a chave é o nome do arquivo e o valor é a matriz correspondente.

O programa apresenta um menu interativo com as seguintes opções\+:
\begin{DoxyItemize}
\item Ler \doxylink{classMatriz}{Matriz}\+: Lê uma matriz a partir de um arquivo.
\item Imprimir \doxylink{classMatriz}{Matriz}\+: Imprime uma matriz armazenada.
\item Somar Matrizes\+: Soma duas matrizes armazenadas.
\item Multiplicar Matrizes\+: Multiplica duas matrizes armazenadas.
\item Sair\+: Encerra o programa. 
\end{DoxyItemize}

Lê dados de um arquivo e insere em uma matriz esparsa. 

Esta função abre um arquivo contendo o número de linhas e colunas de uma matriz, seguido por múltiplas linhas que descrevem itens não nulos da matriz. Cada linha deve conter índices (i, j) e o valor correspondente.


\begin{DoxyParams}{Parâmetros}
{\em matriz} & Referência para o objeto \doxylink{classMatriz}{Matriz} que será inicializado e populado. \\
\hline
{\em filename} & Nome do arquivo a ser lido (sem o caminho completo).\\
\hline
\end{DoxyParams}

\begin{DoxyExceptions}{Exceções}
{\em std\+::runtime\+\_\+error} & Quando não é possível acessar ou abrir o arquivo.\\
\hline
\end{DoxyExceptions}

\begin{DoxyItemize}
\item O arquivo é aberto a partir do diretório "{}src/arquivos/"{} concatenado ao nome do arquivo passado em {\ttfamily filename}.
\item Os primeiros valores lidos do arquivo correspondem ao número de linhas ({\ttfamily linhas}) e de colunas ({\ttfamily colunas}) para inicializar corretamente a matriz.
\item Em seguida, cada conjunto de três valores (índice de linha, índice de coluna e valor) é lido e inserido na matriz usando {\ttfamily matriz.\+insert()}.
\item Caso o arquivo não seja encontrado ou ocorra algum outro problema, é gerada uma exceção do tipo {\ttfamily std\+::runtime\+\_\+error}. 
\end{DoxyItemize}

Salva uma matriz em um mapa associativo de matrizes, permitindo que seja recuperada posteriormente. 

A função solicita ao usuário se deseja salvar a matriz atual. Caso a resposta seja afirmativa, é solicitado um nome para identificar a nova matriz no mapa. Se esse nome já existir, o usuário é informado para tentar novamente com outro nome. Em caso de sucesso, a matriz é inserida no mapa com a chave fornecida pelo usuário, e é exibida uma mensagem de confirmação.


\begin{DoxyParams}{Parâmetros}
{\em matriz} & Objeto do tipo \doxylink{classMatriz}{Matriz} que será salvo. \\
\hline
{\em matrizes} & Estrutura (unordered\+\_\+map) onde a matriz será armazenada, associada a um nome (string).\\
\hline
\end{DoxyParams}
\begin{DoxyNote}{Observação}
Essa função não retorna valores. É importante que o usuário insira corretamente as opções (s ou n) para prosseguir ou cancelar o salvamento, e que forneça um nome válido quando optar por salvar a matriz. 
\end{DoxyNote}
